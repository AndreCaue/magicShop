# Economy & Performance Rules
- Sempre use o modelo mais leve possível (Gemini Flash ou equivalente) para tarefas simples.
- Resuma o histórico a cada 8 interações (não guarde tudo).
- Use apenas as skills necessárias — nunca chame skills extras.
- Sempre responda em JSON estruturado quando possível.
- Antes de executar, planeje em no máximo 3 passos.
- Cacheie resultados de ferramentas repetidas.
- Se a tarefa for simples, finalize em 1 rodada.
